# Scope Upload Parsing as a Beta Path

LawAgent Phase 2 treats pasted text or pasted business scenarios as the stable material-review path. PDF/DOCX upload is supported as a Docling-backed beta path that feeds the same review workflow after parsing, but complex scanned documents, difficult tables, page-level citation quality, and parser-quality benchmarking are not Phase 2 blockers.

This keeps the main product loop focused on compliance review and retrieval evaluation while still reusing the Docling parser already integrated in the data pipeline.
