# Use Scenario Review Cases as the Main Golden Set

LawAgent Phase 2 upgrades evaluation from mostly clause-location questions to scenario review cases. Clause-location cases remain as sanity checks, but the main golden set should cover material-driven data compliance scenarios with expected sources, citation roles, required review facts, second-retrieval expectations, abstention expectations, and materials that must not be cited as clause-level legal basis.

This makes evaluation prove the product loop: extracted review facts and metadata-aware hybrid retrieval must recall the right mix of national law, local or industry conditions, implementation references, and policy explanations for concrete compliance-review scenarios.
